## 🛡️ About This Repository

<img align="right" src="https://media.giphy.com/media/RbDKaczqWovIugyJmW/giphy.gif" width="240" alt="Hacking Animation"/>

Welcome to my personal **TryHackMe (THM)** writeup repository — a living, ever-growing knowledge base that documents my hands-on journey through the world of ethical hacking, penetration testing, and cybersecurity. 🚀

Each room documented here represents **genuine hours of hands-on practice**, research, and methodical problem-solving — translated into clean, structured writeups so that others (and my future self) can learn, reference, and grow from them.

**TryHackMe** is one of the world's most widely adopted platforms for developing offensive and defensive security skills through a gamified, browser-based environment. Rooms span the full spectrum — from beginner-friendly networking fundamentals all the way to sophisticated, real-world penetration testing scenarios.

This repository delivers:

- 📝 **Step-by-step walkthroughs** for every completed room
- 🔍 **Deep-dive explanations** of the *why* behind every technique
- 🛠️ **Tool references and command references** for fast recall
- 📸 **Screenshot evidence** documenting each exploitation step
- 💭 **Personal notes** on what worked, what didn't, and key lessons learned

## 🗺️ Learning Journey

This journey begins at the very foundation — networking fundamentals, Linux command-line proficiency, and basic scripting — and progressively advances through increasingly sophisticated attack and defence scenarios:

```
Foundation ──────────────────────────────────────────────────► Advanced
    │                                                               │
    ▼                                                               ▼
Networking       Web App        Privilege        AI Security    Red Teaming
Linux CLI   ──►  Security  ──►  Escalation  ──►  Research  ──►  (Planned)
Scripting        Crypto         CTF Chains        Forensics
```

**Key Milestones Achieved:**

- ✅ **Web Application Pentesting** — IDOR, SSRF, LFI, SQL Injection, Authentication Bypass
- ✅ **Linux Privilege Escalation** — SUID binaries, cron jobs, misconfigured permissions
- ✅ **Password & Hash Attacks** — MD5, SHA-1, SHA-256, bcrypt cracking with Hashcat & John
- ✅ **Cryptographic Analysis** — XOR cipher identification and plaintext recovery
- ✅ **AI Security Research** — Prompt injection techniques against LLM-based applications
- ✅ **Reconnaissance & Enumeration** — Nmap, Gobuster, ffuf, subdomain enumeration
- ✅ **Exploitation Frameworks** — Metasploit, manual CVE exploitation, searchsploit
- 🔄 **SOC Operations** — Log analysis, incident response (In Progress)
- ⏳ **Active Directory Attacks** — Kerberoasting, BloodHound, lateral movement (Planned)
- ⏳ **Red Teaming** — Advanced adversary simulation (Planned)

## 📖 How to Use This Repository

Whether you're a first-time hacker or a seasoned practitioner looking for a second perspective, here's how to extract maximum value from these writeups:

1. 🎯 **Try the room yourself first** — The best learning comes from genuine struggle. Use these writeups as a guide *after* making a real attempt. Even partial progress is deeply valuable.

2. 🧠 **Read the explanations, not just the commands** — Every writeup explains *why* a technique works, not just *how* to run it. Transferable understanding beats copy-paste hacking.

3. 💻 **Follow along in a live environment** — TryHackMe provides browser-based attack machines. Replicate every step in real time for hands-on reinforcement.

4. 📓 **Take your own notes** — Personalise the material to your learning style. Fork this repository and build on top of it.

5. 🔗 **Cross-reference the tools** — Each writeup links to the tools used. Explore their documentation to deepen your command of each tool.
